
1. Set System Path to contain: C:\WEBs\8020\FormR\_2\bin
   - Click Start, type env, select: Edit the system environment variables
   - Click on button, Environment Variables
   - Under System variables, select Path, then click Edit... 
   - Click on button, New, then type: C:\WEBs\8020\FormR\_2\bin
   - Click Ok three times

2. Get FRTools from suzeeparker GitHub repository
   - Accept Invitation to join suzeeparker/FRTools_dev01-suzee
   - Confirm/create access to your GitHub account: ssh github_btg 
   - Open Command prompt
   - $ cd \WEBs\8020\FormR
   - $ git clone  github_btg:suzeeparker/FRTools_dev01-suzee  frtools

3. Create a link from ./FormR/_2  ./FormR/frtools/_2 to 
   - Open Command prompt
   - $ runas /user:administrator cmd
   - $ cd \WEBs\8020\FormR
   - $ mklink /d _2 frtools\_2

4. Run FRTools command 
   - $ frt
